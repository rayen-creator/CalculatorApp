
# Calculator app

a very basic calculator to learn more about react and it's hooks

## Usage

1-clone the repo 

```
git clone https://github.com/rayen-creator/CalculatorApp
```
2-Install npm packages

 ```sh
$ yarn install
 ```
or

```sh
$ npm install
```

3- run app

```sh
$ yarn start
```
or

```sh
$ npm start
```
